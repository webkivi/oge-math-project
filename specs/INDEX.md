# Индекс спек (вики-слой)

Паттерн: `specs/*_v*.md` — неизменяемые raw sources (новая версия спеки = новый
файл; старый файл не правится и не удаляется, он становится архивом). Этот файл —
поддерживаемая вики-страница поверх них: какая версия сейчас действует, какая
устарела, как файлы связаны друг с другом. Перед чтением спек по задаче — сюда,
это быстрее, чем листать все версии.

Источник правды по тому, что «принято» — Project Brain (Журнал решений). Этот
индекс синхронизируется с ним вручную; если разойдутся — Brain важнее (см.
CLAUDE.md §1).
